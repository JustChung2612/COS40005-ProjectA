# Hướng dẫn chạy project

## 0. Clone Project
``` 
git clone + link 
```

## 1. Install Project

### 1.1 Frontend
```
1. cd frontend 
```
```
2. npm install
```

### 1.2 Backend
```
1. cd backend
```
```
2. npm install
```

## 2. Chạy BE + FE

### 2.1 Frontend

```
1. cd frontend 
```
```
2. npm run dev
```

### 2.2 Backend
```
1. cd backend 
```
```
2. npm start
```
